PROGRAMMING LANGUAGES, DEVELOPMENT TOOLS

From all the programming languages, we chose C# because it's familiar for all of us, and it's a fast multi-platform language, with great extensions!

For the making of the game we will use Godot Engine, which gives us a basic engine to work with, and manages the rendering and in-game updates.

We are writing the code in two different code editors: Visual Studio Code and IntelliJ IDEA, we chose these editors because these are what we use regularly. Finally, for the version control we chose Git and hosted our repository on Github , so we all can reach it from home or at University between classes.
